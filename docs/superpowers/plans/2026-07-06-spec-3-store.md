## Global Constraints

- **Campos Requeridos:** `title` (string) y `content` (string) no deben ser nulos ni vacíos.
- **Valores por defecto:** `status` debe ser `draft` si no se envía en el body.
- **Reglas de BD:** El campo `slug` es `not null` y `unique` en el modelo, por lo que si no se provee en el body, el endpoint deberá autogenerarlo a partir del `title` (usando `slugify` u otra librería/función simple) antes de guardarlo en la base de datos.
- **Salida:** `201 Created` devolviendo el JSON canónico de post (schema `PostRead`).
- **Errores:** `400 VALIDATION_ERROR` estándar proporcionado por el error handler global cuando falten datos obligatorios o el formato del input sea inválido. No redefinir handlers de errores.

```python
import re
from fastapi import APIRouter, Depends, status
from sqlalchemy.orm import Session
from app.database import get_db
from app.models.post import Post
from app.schemas.post import PostCreate, PostRead

router = APIRouter(prefix="/posts", tags=["Posts"])

def generate_slug(title: str) -> str:
    # Lógica simple de slugify o usar un paquete
    slug = title.lower()
    slug = re.sub(r'[^a-z0-9]+', '-', slug)
    return slug.strip('-')

@router.post("/", response_model=PostRead, status_code=status.HTTP_201_CREATED)
def create_post(post_in: PostCreate, db: Session = Depends(get_db)):
    # Crear la instancia del modelo de BD
    db_post = Post(
        title=post_in.title,
        content=post_in.content,
        status=post_in.status,
        slug=generate_slug(post_in.title) # Asignando un slug porque es not null en BD
    )
    db.add(db_post)
    db.commit()
    db.refresh(db_post)
    return db_post
```
